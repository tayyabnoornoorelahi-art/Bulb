## ğŸ“– Usage

Simply open `index.html` in your browser to view all modal examples.

### Opening a Modal

Each modal can be opened by clicking its corresponding button on the main page.

### Closing a Modal

Modals can be closed by:
- Clicking the X button
- Clicking outside the modal
- Pressing the ESC key
- Clicking action buttons (Cancel, Close, etc.)
